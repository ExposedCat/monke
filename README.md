# Monke | AI Pet

## Quick Start
- Install dependencies using `pip install -r requirements.txt`
- Run app using `python src/main.py`

## Models
- Input: **OpenAI Whisper (Medium)**
- Processing: **Meta Llama 3.1**
- Output: **Coqui XTTS 2**

## Known Issues
- Library logs
- Input artefacts (such as "you")
- Can't stop current task
